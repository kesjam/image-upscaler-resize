# Contribution Guidelines

## Development Setup
1. Fork repository
2. Install dependencies:
   ```bash
   npm ci
   ```
3. Create feature branch:
   ```bash
   git checkout -b feat/new-feature
   ```

## Testing
Run unit tests:
```bash
npm test
```

## Code Standards
- TypeScript strict mode
- Prettier formatting
- ESLint rules enforcement
- JSDoc documentation required
